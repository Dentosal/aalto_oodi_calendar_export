Security & privacy

Logging

Passwords are never logged. IP addresses and usernames might be stored in the temporary log files for debugging, but are never stored persistently. IP addresses are also used for rate limitig, but this information is not stored persistently either.

Cookies and tracking

This site does not use cookies, trackers or ads.

Password security

User passwords are never stored on the server, and are always deleted after the content has been fetched from WebOodi. The subscription url contains username and password, but they are AES256-encrypted, so that only this application can decrypt them. Still, fundamentally this application uses your credentials to access WebOodi. If you don't trust that I handle the credentials properly, you are free to run your own instance, simply clone the source from GitHub and upload it to Heroku. It's free, and relatively quick.
